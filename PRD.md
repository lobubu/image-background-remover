# MVP 需求文档：Image Background Remover

> 基于 Next.js + remove.bg API 的背景移除工具

---

## 1. 项目概述

### 1.1 产品定位
一款在线图片背景移除工具，用户上传图片后一键去除背景，生成透明 PNG 下载。

### 1.2 目标用户
- 电商卖家（产品图处理）
- 设计师（素材抠图）
- 普通用户（证件照、头像处理）

### 1.3 核心价值
- **零学习成本**：上传 → 等待 → 下载
- **无需安装**：纯网页操作
- **快速交付**：5 秒内完成处理

---

## 2. 技术架构

```
用户浏览器 → Next.js (Vercel/CF Pages) → remove.bg API
```

### 2.1 技术栈

| 组件 | 技术选型 | 说明 |
|------|----------|------|
| 前端框架 | Next.js 16 | React 全栈框架 |
| 样式 | Tailwind CSS | 原子化 CSS |
| 语言 | TypeScript | 类型安全 |
| 托管 | Vercel / Cloudflare Pages | 免费、全球 CDN |
| 背景移除 | remove.bg API | 成熟稳定、质量高 |

## 4. 非功能需求

### 4.1 性能

| 指标 | 目标值 |
|------|--------|
| 页面首次加载 | < 2s |
| 图片上传（5MB） | < 3s |
| 背景移除处理 | < 5s |
| 整体响应时间 | < 10s |

### 4.2 限制

| 限制项 | MVP 阶段 | 说明 |
|--------|----------|------|
| 单张图片大小 | ≤ 10MB | remove.bg 限制 |

| 支持格式 | JPG, PNG, WebP | 常见图片格式 |

---

## 5. 接口设计

### 请求

```
POST /api/remove-background
Content-Type: application/json
{ "image": "data:image/jpeg;base64,..." }
```

### 成功响应

```json
{ "success": true, "result": "data:image/png;base64,..." }
```

### 错误码

| 错误码 | HTTP 状态码 | 说明 |
|--------|-------------|------|
| INVALID_REQUEST | 400 | 请求格式错误 |
| IMAGE_TOO_LARGE | 400 | 图片超过 10MB |
| UNSUPPORTED_FORMAT | 400 | 不支持的图片格式 |
| RATE_LIMIT_EXCEEDED | 429 | 请求过于频繁 |
| API_ERROR | 500 | remove.bg API 调用失败 |
| QUOTA_EXCEEDED | 503 | 当日配额已用完 |

# MVP 需求文档：Image Background Remover

> 基于 Cloudflare Workers + remove.bg API 的背景移除工具

---

## 1. 项目概述

### 1.1 产品定位
一款在线图片背景移除工具，用户上传图片后一键去除背景，生成透明 PNG 下载。

### 1.2 目标用户
- 电商卖家（产品图处理）
- 设计师（素材抠图）
- 普通用户（证件照、头像处理）

### 1.3 核心价值
- **零学习成本**：上传 → 等待 → 下载
- **无需安装**：纯网页操作
- **快速交付**：5 秒内完成处理

---

## 2. 技术架构

```
┌─────────────────────────────────────────────────────────────┐
│                        用户浏览器                            │
│  ┌─────────────┐    ┌─────────────┐    ┌─────────────┐     │
│  │  上传图片   │ → │  预览处理   │ → │  下载结果   │      │
│  └─────────────┘    └─────────────┘    └─────────────┘     │
└───────────────────────────┬─────────────────────────────────┘
                            │ HTTPS
                            ▼
┌─────────────────────────────────────────────────────────────┐
│                  Cloudflare Pages (前端静态托管)             │
│  - HTML/CSS/JS                                              │
│  - 响应式设计                                                │
│  - 拖拽上传、进度条、预览对比                                 │
└───────────────────────────┬─────────────────────────────────┘
                            │ Fetch API
                            ▼
┌─────────────────────────────────────────────────────────────┐
│                  Cloudflare Workers (API 网关)               │
│  - 请求验证                                                  │
│  - 图片大小/格式校验                                          │
│  - 调用 remove.bg API                                        │
│  - 错误处理 & 日志                                            │
│  - 访问频率限制                                               │
└───────────────────────────┬─────────────────────────────────┘
                            │ HTTPS API
                            ▼
┌─────────────────────────────────────────────────────────────┐
│                     remove.bg API                            │
│  - AI 背景移除                                                │
│  - 返回透明 PNG                                               │
└─────────────────────────────────────────────────────────────┘
```

### 2.1 技术栈

| 组件 | 技术选型 | 说明 |
|------|----------|------|
| 前端 | HTML + Tailwind CSS + Vanilla JS | 轻量、快速加载 |
| 前端托管 | Cloudflare Pages | 免费、全球 CDN |
| API 网关 | Cloudflare Workers | 边缘计算、免费额度充足 |
| 背景移除 | remove.bg API | 成熟稳定、质量高 |

### 2.2 数据流

1. 用户在前端上传图片
2. 前端校验文件大小/格式，转为 Base64
3. 前端调用 Workers API
4. Workers 调用 remove.bg API
5. remove.bg 返回处理后的 PNG
6. Workers 返回给前端
7. 前端展示预览，提供下载

**注意**：全程内存处理，不落盘存储。

## 4. 非功能需求

### 4.1 性能

| 指标 | 目标值 |
|------|--------|
| 页面首次加载 | < 2s |
| 图片上传（5MB） | < 3s |
| 背景移除处理 | < 5s（取决于 remove.bg） |
| 整体响应时间 | < 10s |

### 4.2 限制

| 限制项 | MVP 阶段 | 说明 |
|--------|----------|------|
| 单张图片大小 | ≤ 10MB | remove.bg 限制 |
| 单日处理量 | ≤ 50 张 | remove.bg 免费额度 |
| 并发请求 | 5 QPS | 防滥用 |
| 支持格式 | JPG, PNG, WebP | 常见图片格式 |

### 4.3 安全

- **API Key 保护**：remove.bg API Key 仅存储在 Workers 环境变量
- **CORS 限制**：仅允许指定域名调用 API
- **请求频率限制**：IP 维度限流，防止滥用
- **文件校验**：后端二次校验文件类型，防止伪装攻击

### 4.4 兼容性

| 平台 | 支持范围 |
|------|----------|
| 桌面浏览器 | Chrome 90+, Firefox 88+, Safari 14+, Edge 90+ |
| 移动浏览器 | iOS Safari 14+, Chrome Android 90+ |
| 屏幕尺寸 | 320px - 2560px 响应式 |

## 5. 接口设计

### 5.1 前端 → Workers API

**请求**

```http
POST /api/remove-background
Content-Type: application/json

{
  "image": "data:image/jpeg;base64,/9j/4AAQSkZJRg..."
}
```

**成功响应**

```http
HTTP/1.1 200 OK
Content-Type: application/json

{
  "success": true,
  "result": "data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAA..."
}
```

**错误响应**

```http
HTTP/1.1 400 Bad Request
Content-Type: application/json

{
  "success": false,
  "error": "IMAGE_TOO_LARGE",
  "message": "图片大小不能超过 10MB"
}
```

### 5.2 错误码

| 错误码 | HTTP 状态码 | 说明 |
|--------|-------------|------|
| INVALID_REQUEST | 400 | 请求格式错误 |
| IMAGE_TOO_LARGE | 400 | 图片超过 10MB |
| UNSUPPORTED_FORMAT | 400 | 不支持的图片格式 |
| RATE_LIMIT_EXCEEDED | 429 | 请求过于频繁 |
| API_ERROR | 500 | remove.bg API 调用失败 |
| QUOTA_EXCEEDED | 503 | 当日配额已用完 |

## 6. 成本估算

### 6.1 免费额度

| 服务 | 免费额度 | 超出费用 |
|------|----------|----------|
| Cloudflare Pages | 无限 | - |
| Cloudflare Workers | 10 万次请求/天 | $0.50/百万请求 |
| remove.bg API | 50 张/月 | $0.20 - $0.10/张（按量递减） |

### 6.2 MVP 阶段成本

- **月处理量 ≤ 50 张**：**$0/月**
- **月处理量 500 张**：约 $90/月
- **月处理量 5000 张**：约 $450/月（单价降至 $0.09/张）

### 6.3 成本优化策略

当日均处理量 > 50 张时，考虑：
1. 切换到 remove.bg 付费套餐（量大折扣）
2. 自建后端使用 rembg / background-remove skill
3. 混合方案：免费额度用 remove.bg，超出走自建

## 8. 风险与应对

| 风险 | 影响 | 概率 | 应对措施 |
|------|------|------|----------|
| remove.bg API 限流 | 用户无法使用 | 中 | 实现请求排队，引导用户稍后重试 |
| 免费额度用尽 | 当月无法继续 | 高 | 监控用量，提前预警；准备付费账号 |
| 图片质量差 | 处理效果不佳 | 中 | 提示用户上传高质量图片 |
| 大量滥用请求 | 成本失控 | 低 | IP 限流 + CAPTCHA |
